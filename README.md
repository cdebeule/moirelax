## moirelax

This repository contains Julia code for computing structural relaxation in moiré bilayers with continuum elasticity theory [1, 2].

It also contains LAMMPS [3] configuration files and data used in Ref. [1].

### References

[1]: [Theory for Lattice Relaxation in Marginally Twisted Bilayers, Christophe De Beule, Gayani N. Pallewela, Mohammed M. Al Ezzi, Liangtao Peng, E. J. Mele, and Shaffique Adam, arXiv:2503.19162 (2025).](https://arxiv.org/abs/2503.19162)

[2]: [Lattice Relaxation in Moiré Heterobilayers, Christophe De Beule, Yiyang Lai, Liangtao Peng, Daniel Bennett, and Shaffique Adam, arXiv:2605.17805 (2026).](https://arxiv.org/abs/2605.17805)

[3]: [LAMMPS Molecular Dynamics Simulator.](https://www.lammps.org)
